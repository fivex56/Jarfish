⚡️ Как запустить своего Джарфиша за 5 минут

Шаг 1. Установи нужные библиотеки

Открой терминал и выполни одну команду — она поставит всё необходимое для бота, распознавания голоса и связи с Google Календарём.

Шаг 2. Создай бота в Telegram

Найди @BotFather, напиши /newbot, придумай имя и логин (должен заканчиваться на bot). В ответ придёт токен — длинная строка, это как пароль от твоего бота.

Шаг 3. Узнай свой Telegram ID

Найди @userinfobot, напиши ему что угодно — он скажет твой ID (число). Оно нужно чтобы бот слушался только тебя.

Шаг 4. Настрой ключи

Переименуй .env.example в .env, открой в блокноте и впиши:
• Токен от BotFather
• Свой Telegram ID
• Ключ DeepSeek (если есть — без него бот не понимает обычную речь, но команды работают)

Где взять ключ DeepSeek: зарегистрируйся на platform.deepseek.com, пополни баланс (100 рублей хватит на месяцы), создай ключ в разделе API Keys.

Шаг 5. Запусти

python jarvis_bot.py

Напиши /start своему боту. Готово.

Совет: чтобы бот сам перезапускался при ошибках — используй launch.bat (Windows) вместо прямого запуска.

## 📦 Что к чему

```
Things/
├── jarvis_bot.py            ← сердце: запуск, расписание, мост CLI↔TG
├── config.py                ← читает .env
├── launch.bat               ← авто-перезапуск при падении
│
├── db/                      ← SQLite: таблицы, миграции, запросы
├── bot/                     ← Telegram-часть: команды, кнопки, меню
├── cli/                     ← Консольный мост
├── services/                ← Мозги: NL-парсер, календарь, напоминалки, речь, картинки
└── utils/                   ← Разбор времени (русский язык)
```

/notes              Заметки
