## 🧠 Что он умеет

| Возможность | Как работает |
|-------------|-------------|
| 📋 Задачи и проекты | Добавляй, выполняй, переноси — голосом, текстом, как угодно |
| ⏰ Напоминания | «Напомни позвонить завтра в 15:00» — и точка |
| 📝 Заметки и лента мыслей | Пиши, наговаривай, фоткай — сохранит и найдёт |
| 🗣 Понимает речь | Голосовое → текст → действие. Без рук |
| 📸 Понимает фото | Присылаешь картинку — описывает, сохраняет в заметки |
| 🌅 Утренний брифинг | Каждое утро: что сегодня, что просрочено, что горит |
| 🌙 Вечерний обзор | Что сделано, что нет. Без прикрас |
| 🔄 Авто-перенос задач | Незавершённое с сегодня — само переедет на завтра |
| 🧙‍♂️ Пацанская мудрость | Каждое утро в 9:00 — философия улиц |
| 💡 Идеи дня | Анализирует твою активность и предлагает улучшения |
| 📅 Google Календарь | События создаются и синхронизируются с твоим календарём |
| 🖥 Консольный мост | Пишешь в консоли — приходит в Telegram, и обратно |

## ⚡ Быстрый старт

```bash
# 1. Установка
pip install "python-telegram-bot[job-queue]>=21.9" aiosqlite httpx openai-whisper google-api-python-client google-auth-oauthlib colorama

# 2. Копируй шаблон и впиши свои ключи
cp .env.example .env

# 3. Погнали
python jarvis_bot.py
```

Всё. Бот живёт в Telegram, пишет в консоль, управляется с двух сторон.

---

## 🔐 Безопасность

В репозитории **нет ни одного ключа или пароля**.

Всё чувствительное лежит только у тебя:
```
.env                    ← ключи (не коммитить!)
oauth_google.json       ← Google OAuth (не коммитить!)
google_token.pickle     ← токен календаря (не коммитить!)
jarvis.db               ← база данных (не коммитить!)
```

Смотри `.gitignore` — там расписано что не улетит в интернет.

## 📦 Что к чему

```
Things/
├── jarvis_bot.py            ← сердце: запуск, расписание, мост CLI↔TG
├── config.py                ← читает .env
├── launch.bat               ← авто-перезапуск при падении
│
├── db/                      ← SQLite: таблицы, миграции, запросы
├── bot/                     ← Telegram-часть: команды, кнопки, меню
├── cli/                     ← Консольный мост
├── services/                ← Мозги: NL-парсер, календарь, напоминалки, речь, картинки
└── utils/                   ← Разбор времени (русский язык)
```
