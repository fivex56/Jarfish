## ⚡ Как запустить своего

Твой собственный Джарфиш за 5 минут:

### Шаг 1. Установи нужные библиотеки

Открой терминал (командную строку) и выполни:

```bash
pip install "python-telegram-bot[job-queue]>=21.9" aiosqlite httpx openai-whisper google-api-python-client google-auth-oauthlib colorama
```

Это поставит всё необходимое для работы бота, распознавания голоса и связи с Google Календарём.

### Шаг 2. Создай бота в Telegram

1. Открой Telegram и найди **@BotFather**
2. Напиши ему `/newbot`
3. Придумай имя (например, *Мой Джарфиш*) и логин (должен заканчиваться на `bot`, например *my_jarfish_bot*)
4. В ответ BotFather пришлёт **токен** — длинную строку вроде `8083988805:AAHGtXbY...`
5. Скопируй этот токен

### Шаг 3. Узнай свой Telegram ID

1. Найди в Telegram бота **@userinfobot**
2. Напиши ему что угодно
3. Он ответит твоим ID — числом вроде `591695423`
4. Запомни его

### Шаг 4. Настрой ключи

Открой файл `.env.example` в папке проекта и сохрани его как `.env` (просто убери `.example` из названия). Затем открой `.env` в блокноте и впиши свои данные:

```
BOT_TOKEN=сюда_вставь_токен_от_BotFather
ALLOWED_USER_ID=сюда_вставь_свой_telegram_id
DEEPSEEK_API_KEY=sk-твой_ключ_deepseek   (если есть, иначе оставь пустым)
```

Где взять ключ DeepSeek:
1. Зайди на [platform.deepseek.com](https://platform.deepseek.com)
2. Зарегистрируйся и пополни баланс (хватит 100 рублей на месяцы работы)
3. В разделе API Keys создай ключ и скопируй его

Без DeepSeek бот тоже будет работать — просто не сможет понимать обычную речь. Команды (`/task_add` и т.д.) работают всегда.

### Шаг 5. Запусти

```bash
python jarvis_bot.py
```

Готово. Напиши `/start` своему боту в Telegram — он ответит.

💡 **Совет:** если хочешь чтобы бот не падал при ошибках и сам перезапускался — используй `launch.bat` (Windows) или настрой systemd-сервис (Linux).

## 🔐 Безопасность

В репозитории **нет ни одного ключа или пароля**.

Всё чувствительное лежит только у тебя:
```
.env                    ← ключи (не коммитить!)
oauth_google.json       ← Google OAuth (не коммитить!)
google_token.pickle     ← токен календаря (не коммитить!)
jarvis.db               ← база данных (не коммитить!)
```

Смотри `.gitignore` — там расписано что не улетит в интернет.
