### 🛑 Хватит имитировать жизнь. Другой не будет.

Твои боты — это просто кнопки. **Jarfish** — это мозги. Первая думающая прослойка между пиздецом в голове и идеальным порядком в делах. Он *понимает*, что ты несёшь голосом, и разгребает это сам.

> **«Бабки капают — не ссы в очко, пятихуем горизонт не замутишь.».**

## 🔥 Что умеет Jarfish

### Задачи и проекты
* Создание задач голосом, текстом или командой — с приоритетом, дедлайном и тегами
* Проекты: активные, на паузе, завершённые — с приоритетами
* Авто-перенос незавершённых задач на следующий день (полночь)
* Умное перепланирование просроченных задач — с кнопками и авто-отменой для вечно висящих

### Напоминания
* Естественный язык: «напомни завтра в 15:00 позвонить»
* Повторяющиеся: daily, weekly — с адаптивным интервалом (сделал — реже, забил — чаще)
* Планировщик без внешних зависимостей (PTB JobQueue)

### Заметки и лента мыслей
* Текст, голос, фото — всё сохраняется и распознаётся
* Голос → Whisper (tiny, локально, без интернета)
* Фото → DeepSeek Vision (распознавание текста и образов)
* Лента мыслей с навигацией по датам и тегам

### Проактивные уведомления
* 08:00 — утренний брифинг: задачи на сегодня, напоминания, просрочки
* 21:00 — вечерний обзор: что сделано, что осталось
* Каждые 4 часа — проверка просроченных задач с умным автопереносом
* Проверка неактивных проектов (>7 дней)

### Google Календарь
* Просмотр событий через OAuth 2.0
* Авто-создание задач из событий календаря

### CLI-мост
* Консоль ↔ Telegram: пиши в консоли — приходит в ТГ и наоборот
* Все команды доступны из командной строки

### Пацанская мудрость
* Каждый день в 9:00 — свежая мудрость от DeepSeek
* Авто-обновление цитаты в этом README и пуш на GitHub

### Саморазвитие (AI Agents)
* **Генератор** (DeepSeek): анализирует проект, ищет тренды, придумывает улучшения
* **Разработчик** (Claude Code CLI): читает код, рецензирует идеи, внедряет изменения в реальные файлы
* Цикл каждые 2 дня + еженедельный стратегический обзор
* Сам коммитит и пушит на GitHub

### Безопасность
* Доступ только для одного пользователя (user_id)
* Чувствительные данные в .env, база данных в .gitignore
* OAuth-токены не коммитятся

## ⚡️ Установка

```bash
git clone https://github.com/fivex56/Jarfish.git && cd Jarfish
pip install -r requirements.txt
```

Создай `.env`:
```
BOT_TOKEN=токен_от_BotFather
ALLOWED_USER_ID=твой_telegram_id
DEEPSEEK_API_KEY=ключ_deepseek
```

Запуск: `python jarvis_bot.py`
